Chapter 1 – Introduction to Data Visualization

1. Define Data Visualization. Why is it important?

Data Visualization is the graphical representation of data using charts, graphs, maps, and dashboards to make information easy to understand.

Importance:

  • Helps identify trends and patterns quickly.
  • Improves decision making in businesses.
  • Makes complex data understandable.
  • Used in AI/ML, business analytics, healthcare, finance, education, etc.
  • Supports predictive analysis.
Raw Data → Cleaning → Analysis → Visualization → Decision Making

2. What charts can represent relationships?

Chart Use Case
Scatter Plot Relationship between advertising and sales
Bubble Chart Population vs GDP vs Area
Heatmap Correlation matrix
Line Chart Temperature change over time

3. Benefits of Data Visualization

  • Quick understanding of large datasets
  • Easy communication of insights
  • Improves productivity
  • Supports strategic planning
  • Detects anomalies and outliers
  • Interactive dashboards improve user experience

4. Categories of Graphs

Category Examples
Comparison Bar Chart, Column Chart
Distribution Histogram, Box Plot
Relationship Scatter Plot, Bubble Chart
Composition Pie Chart, Stacked Bar Chart
Time Series Line Graph, Area Chart

5. Graphs for Part-to-Whole Relationships

  • Pie Chart: Shows percentage contribution.
  • Donut Chart: Similar to pie chart but modern style.
  • Stacked Bar Chart: Comparison and composition together.
  • Treemap: Hierarchical part-to-whole relationship.

6. Visualizations for Flows and Processes

Visualization Application
Flowchart Algorithm design
Sankey Diagram Energy transfer
Process Map Business workflow
Network Diagram Social network analysis

7. Charts for Data Distribution

  • Histogram: Frequency distribution.
  • KDE Plot: Smooth probability density curve.
  • Box Plot: Median, quartiles, outliers.
  • Violin Plot: Distribution + density.

8. Graphs for Comparing Values

9. Visualizations for Relationships

Chapter 2 – Matplotlib

1. Company Revenue Over Years

import matplotlib.pyplot as plt

years = [2015,2016,2017,2018,2019,2020,2021,2022,2023,2024]
revenue = [5,7,8,10,12,14,15,18,20,22]

plt.plot(years, revenue, 'r--', linewidth=2)
plt.xlabel('Years')
plt.ylabel('Revenue in Million Dollars')
plt.title('Company Revenue Over Years')
plt.grid(True)
plt.show()

2. Difference between Bar Chart and Histogram

Bar Chart Histogram
Used for categorical data Used for continuous numerical data
Bars separated by spaces Bars touch each other
Compares categories Shows distribution

3. Histogram of Student Marks

import numpy as np
import matplotlib.pyplot as plt

marks = np.random.randint(0, 101, 50)

plt.hist(marks, bins=10, color='skyblue', edgecolor='black')
plt.xlabel('Marks Range')
plt.ylabel('Number of Students')
plt.title('Histogram of Student Marks')
plt.grid(True)

4. Scatter Plot for Advertising Budget vs Sales

import matplotlib.pyplot as plt
import numpy as np

budget = [10,20,30,40,50,60]
sales = [15,25,35,45,55,70]

plt.scatter(budget, sales, color='blue', marker='o')

m, b = np.polyfit(budget, sales, 1)
plt.plot(budget, m*np.array(budget)+b, color='red')

plt.xlabel('Advertising Budget ($)')
plt.ylabel('Sales Revenue ($)')
plt.title('Advertising vs Sales')
plt.show()

5. Pie Chart – Smartphone Market Share

import matplotlib.pyplot as plt

brands = ['Apple', 'Samsung', 'Xiaomi', 'Oppo']
share = [35, 30, 20, 15]
explode = [0.1, 0, 0, 0]
colors = ['gold', 'lightblue', 'lightgreen', 'pink']

plt.pie(share,
        labels=brands,
        autopct='%1.1f%%',
        explode=explode,
        colors=colors)

plt.title('Smartphone Market Share 2024')
plt.show()
