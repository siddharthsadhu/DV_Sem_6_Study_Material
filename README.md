# 🌟 Deployment

This project is deployed using **GitHub Pages**.

## Steps:

1. Go to Repository Settings
2. Open "Pages"
3. Select:
   - Branch → `main`
   - Folder → `/root`
4. Save

GitHub will generate your website link automatically.
